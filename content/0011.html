実は自由性が高いサイト

一応だが本ブログは廃止されたプロジェクトであるインターネット百科事典「Elexandria Library」をベースとして作ったため、記事サンプルを見れば分かるように無駄なレベルで文章が装飾できるように作られている。これから心機一転して真面目な記事を書くことはないと思うが、せっかくなので何となくMathJaxを導入することにした。あれ、これ導入する意味あるのか？

MathJaxとは？

MathJaxはウェブ上に数式を表示するオープンソースのシステム…云わばJavaScriptライブラリーである。サーバー側が用意していればクライアント側の事前設定・導入操作などの必要は皆無、ブラウザーの種類を問わずに高度な数式を見ることができるのだ。また画像を貼り付けるわけではないため、どれだけ拡大してもボヤボヤしたジャギーが出ることはない。ちなみにMarkdown形式の中でも(大抵は)使える。

大学生になれば論文など何処かしらでこの書式を使うため、今のうちに慣れておいて損はない。

導入手順

うん、とりあえず<head>と</head>の間に 
<script
    type="text/javascript"
    src="https://cdnjs.cloudflare.com/ajax/libs/mathjax/2.7.0/MathJax.js?config=TeX-AMS_CHTML">
</script>
と 
<script type="text/x-mathjax-config">
    MathJax.Hub.Config({
        tex2jax: {
            inlineMath: [ ['$','$'], ["\\(","\\)"] ],
            displayMath: [ ['$$','$$'], ["\\[","\\]"] ]
        }
    });
</script>
をぶち込めば使えるようになる。簡単に説明すれば1番目のコードでMathJaxを導入するためのスクリプトをインターネット上から引っ張り出して、2番目のコードでHTMLのボディー内にあるMathJax用の書式を変換する感じ。やっぱり現代って神だわ。

実際に書いてみよう

たとえばマンデルブロ集合の定義を書くならHTML内に 
\[
    \begin{cases}Z_{n+1}=z_{n}^{2}+c\\ Z_{n}=0\end{cases}
\]
と書けばあら不思議！ \[\begin{cases}Z_{n+1}=z_{n}^{2}+c\\ Z_{n}=0\end{cases}\] はい、皆大好きマンデルブロ集合の定義が出せました。

ちなみに、例えば\(1:\dfrac{1+\sqrt{5}}{2}\)とか文章内に黄金比を書く場合は 
\(
    1:\dfrac{1+\sqrt{5}}{2}
\)
のように\[\]を\(\)へ変えるだけ。どう、簡単でしょ？

と、こんな感じに数式であれば何でも書くことができる。勢いで書いてしまったが、肝心の中身(記号の説明とか書式の理解とか)を紹介すると終がないため、もしもMathJaxで色々と書きたい方はEasy Copy MathJaxというMathJaxのコマンド一覧を参考にしながら書いてほしい。…マジで便利。

え？一々訳の分からない記号と睨めっこするのが嫌だって？仕方ないなぁ、のび太君は…。ほら、MathJax Supporter : Web Equation Mathをあげるよ。これは数式を手書きするだけでMathJax形式に変換してくれたり、表示サンプルやグラフで可視化してくれる神ツールなんだ。…ガチで便利。
